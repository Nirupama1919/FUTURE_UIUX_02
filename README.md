# FUTURE_UIUX_02
# GlowUp Salon Booking App

A modern mobile-first salon appointment booking application designed in Figma as part of the Future Interns UI/UX Design Task 2 (2026).

---

## 📌 Project Overview

GlowUp is a salon booking app UI designed to simplify appointment scheduling for salon customers.

The app allows users to:
- Explore salon services
- View service details
- Select date & time slots
- Book appointments easily
- Receive booking confirmation

The project focuses on clean UI, smooth UX flow, and mobile usability.

---

## 🎯 Problem Statement

Many local salons still rely on:
- Phone calls
- WhatsApp bookings
- Manual appointment registers

This often creates:
- Scheduling confusion
- Missed appointments
- Poor customer experience
- Inefficient booking management

GlowUp solves this by providing a modern digital booking experience.

---

## 👥 Target Users

- Salon customers aged 18–35
- Busy professionals and students
- Users preferring online booking
- First-time mobile app users
